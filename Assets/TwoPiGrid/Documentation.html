BaseGrid

public class BaseGrid

Fields and properties

public int CellCount

The amount of cells the grid has.

public Vector3 Center

The grid's center in world coordinates. Can be changed dynamically.

public float Radius

The grid's radius. Can be changed dynamically.

Constructors

public BaseGrid(string settingsFileName)

Construct the grid by passing it its settings object file name. If the settings can't be found, a default grid will be created instead (12 cells, radius of 1, center at (0, 0, 0)).

  • string settingsFileName
    Name of a GridSettingsSerialized asset that should be in a Resources folder.

public BaseGrid(IGridSettings settings)

Construct the grid by passing it an IGridSettings object.

  • IGridSettings settings
    Settings object.

Methods

public Vector3 GetCellPosition(int i)

Query the position of a cell (in world coordinates).

  • int i
    Index of the cell.

Returns: The position of the center of the cell.

public int GetIndexOfClosestVertexTo(Vector3 point)

Gets the index of the grid's cell that is closest to the given position in world coordinates.

  • Vector3 point
    Position in world coordinates to query.

Returns: The index of the closest cell to the point.
